--- layout: project_details project_name: E-commerce image1: assets/img/portfolio/project4/app_screenshot.png image2: assets/img/portfolio/project4/app_screenshot2.png stack: Reacj.js, Redux, Material UI/MUI github_link: https://github.com/wandergithub/e-commerce --- Completely responsive E-commerce web page. build with React/Redux and MaterialUI-styled-components. This allows the user to navigate through different categories of items, filter them by category and add them to the shopping cart. Users can manage their current bag on the shopping cart containing a list of items where each one can be deleted. In the shopping cart, the user can delete or finish buying.
